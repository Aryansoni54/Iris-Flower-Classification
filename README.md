# Iris-Flower-Classification
🌸 Iris Flower Classification using Machine Learning

This project demonstrates an end-to-end machine learning workflow to classify iris flowers into three species: Setosa, Versicolor, and Virginica using the built-in Iris dataset from Scikit-learn.

Project Highlights

Data Loading & Exploration: Understand dataset features and target classes.

Preprocessing & Scaling: Standardized features for better model performance.

Model Training: Built classification models using Random Forest and Logistic Regression.

Model Evaluation: Assessed models using accuracy, confusion matrix, and classification report.

Feature Importance: Identified which features (petal length, petal width, etc.) most influence predictions.

Model Saving: Trained Random Forest model saved using Pickle for future deployment.

Results

Random Forest achieved the highest accuracy (~97%), outperforming Logistic Regression (~95%).

Key predictive features: petal length and petal width.

Impact

This project provides a practical example of supervised learning and can help in educational or portfolio projects, showcasing model building, evaluation, and deployment.
